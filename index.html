Data Processing & CI/CD Workflow

This project demonstrates an automated data processing pipeline using Python, Pandas, Ruff, and GitHub Actions. It takes an Excel file (data.xlsx), converts it to CSV, processes it with a Python script (execute.py), and publishes the resulting JSON output to GitHub Pages.

Project Goal

  • Fix and execute a Python script reliably.
  • Automate data conversion from XLSX to CSV.
  • Implement a robust CI/CD pipeline with GitHub Actions.
  • Publish generated output using GitHub Pages.

How It Works

  1. Data Preparation: data.xlsx is converted to data.csv.
  2. Script Execution: The fixed execute.py script processes data.csv.
  3. Output Generation: The script's output is redirected to result.json.
  4. CI/CD Automation: A GitHub Actions workflow handles linting with Ruff, data conversion, script execution, and deployment.
  5. Web Publishing: result.json is published via GitHub Pages.
